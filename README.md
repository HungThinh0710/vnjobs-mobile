## Installation Guide
1. Clone This project.

	`git clone https://github.com/SinJunior/vnjobs-mobile.git`
2. Install necessary components with yarn or npm.

	`cd vnjobs-mobile\ `

	`yard` or `npm install`
3. Run App with yarn or npm

  	`yarn start` or `npm start`
